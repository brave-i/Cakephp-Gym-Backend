Documented by Mojoomla - at October 20 2016

A) Introduction

Gym Management System for Cake php is ideal way to manage complete Gym operation. It has different user roles like Staff Member, Member, accountant and admin user.

This user guide covers all options that this cake php scripts provides. Please read the documentation carefully as it probably contains answers to most of questions that may arise. If you feel some aspect is not covered or have some questions, contact us via e-mail sales@mojoomla.com. We will do our best to answer you in shortest time.

CakePHP 3 Requirement

  1. HTTP Server. For example: Apache. Having mod_rewrite is preferred, but by no means required.
  2. PHP 5.5.9 or greater[5.6 Recommended]
  3. mbstring PHP extension.
  4. intl PHP extension.

Please confirm above requirements to your host for smooth installation process.

5) Update :

No need to follow below steps if you are installing first time.

Migration to Verion 2.0 for already installed gym master application

Follow the steps to update system

  1. Extract update_v2 folder from gym_mater.zip file
  2. Open the extracted update_v2 folder from the gym_master.zip file.
  3. Copy src and webroot folders files and replace it in already installed src and webroot folder in gym_master directory.
  4. NOTE : PLEASE BACKUP YOUR SRC AND WEBROOT FOLDER BEFORE UPDATING.

Mobile API :

  1. Mobile API files are located at "gym_master/webroot/nghome" folder
  2. To Make connection between your "Web application and Mobile App",Please update your DB details in "gym_master/webroot/nghome/connection.php"
  3. NOTE: Do not change directory name or location.Otherwise API will stop working.
